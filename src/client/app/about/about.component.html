About this application

This is an exercise in Angular2.

It is built with Angular2, Typescript, and uses Bootstrap for the UI.

The app has the following behavior:

  • The user signs in using Facebook credentials. Then the user is redirected to a geolocation page, which displays the user's location on a Google Map.
  • The geolocation page is protected behind an authentication guard. So, it can be accessed only when signed in.
  • The about page is open for public view. And can be viewed without signing in.
  • The switch theme button on the top right corner allows switching between two simple themes.

For miscellaneous details, please refer to the README file in the github project.
